WELCOME TO THE JOB DESCRIPTION BUILDER

Create JDX-Compliant Job Descriptions

Job Description Builder helps you create job descriptions that align with established skill and competency standards. You can also create and manage job openings that can be published in conformance with the U.S.Chamber of Commerce Foundation's Job Data Exchange (JDX) project.

You can:

  • upload your existing job descriptions
  • match skills and competencies with established standards
  • export job descriptions and openings in JDX-approved JSON formats
  • help us improve this experience and advance job placements
